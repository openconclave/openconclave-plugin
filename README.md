# OpenConclave Plugin for Claude Code

AI agent orchestration with visual workflow automation.

## Prerequisites

Install OpenConclave first:

**macOS / Linux:**
```bash
curl -fsSL https://openconclave.com/install.sh | bash
```

**Windows:**
```powershell
irm https://openconclave.com/install.ps1 | iex
```

## Install Plugin

```
/plugin marketplace add openconclave/openconclave-marketplace
/plugin install openconclave
```

## What You Get

- MCP tools for managing workflows from Claude Code
- Channel events for workflow output and prompt questions
- Auto-start server on session start
- `/create-workflow` skill
- `/openconclave` command
